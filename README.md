## 技术栈

前端技术选型，Vue 3，Vue Route，Vuex，等vue系列负责基本可视化页面的展示，Axios，负责数据传输，Bootstrap负责优化页面显示。

后端技术选型，Spring Boot构成后端基本框架，MyBatis-Plus依赖简化数据库交互逻辑，Netty，websocket作为即时通信，也集成在后端；JSON Web Token（简称JWT）负责用户信息认证。

数据库与存储，采用MySQL数据库，对象存储和文件管理MinIO，数据库通过链接访问文件。
